Prerequisites

The workshop is open to anyone with no (or little) prior programming experience. Attendees must however feel comfortable enough with their own computer to install programs (and debug their installation if needed) and locate folders and files.

Learning objectives

Attendees will gain a high-level understanding of data import, manipulation and graphing with R and RStudio. They will be able to produce a variety of publication-quality graphs (among others: boxplots, histograms, dot plots, heatmaps).

Learning outcomes

  • Identify and use RStudio panels (console, scripts, folders and files panels).
  • Locate useful resources to learn more and know where to seek help.
  • Import data from files into R.
  • Manipulate and prepare (filter, select) data.
  • Produce graphs:
    • Create a plot from “recipes”.
    • Change parameters (color, font size, point shape, title, etc.).
    • Export high-quality graphs in different formats (pdf, jpeg, png).
